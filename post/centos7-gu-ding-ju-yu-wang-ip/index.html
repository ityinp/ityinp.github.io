centos7 固定局域网 ip

  1. 2021-07-30
  2. 229字
  3. 1 min read

  1. 首先查看自己的 ip 是什么：

    发现有两个网卡设备，其中 eno33 是我们的有线网卡，ip 地址是 192.168.7.29
  2. 控制台输入：
netstat -rn

来查看子网掩码跟网关

3. 得到需要的信息后，更改网络配置文件：
进入如下路径

/etc/sysconfig/network-scripts/

找到与自己网卡名字相同的 ifcfg-xxxx 文件打开吧编辑
列如：我的是 ens33, 我就打开下图的文件

一下内容文档中没有的加进去，有的修改即可（子网掩码、默认网关设置成未固定 ip 时查询出来的，本机地址固定成想要的 ip）

BOOTPROTO="static"#静态IP
IPADDR=192.168.7.27#本机地址
NETMASK=255.255.255.0 #子网掩码
GATEWAY=192.168.7.254 #默认网关
DNS1=114.114.114.114
ONBOOT="yes"#开机启动
  1. 重启网络：
service network restart
Restarting network (via systemctl):                        [  OK  ]
